# DeciDeFi

A Defi and Coin Tracker created using Bootstrap, React and CoinGecko API.


## Features
- Bootstrap
- React Router
- Responsive Design
- Carousel on homepage and Dex page

 - [React Bootstrap](https://react-bootstrap.netlify.app/)
